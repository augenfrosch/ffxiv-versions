# ffxiv-versions

List of FFXIV versions since 7.0 for the Global/JP, CN, KR, and TW/TC releases.

Currently, the data is available as [CSV](./data/csv/) or [JSON](./data/json/), although other formats that have an implentation for [Serde](https://serde.rs/) would be easy to add to [`ffxiv-versions-re-export`](./re-export/).

> [!IMPORTANT]
> The data is added manually. The data updates are therefore delayed from when the version/patches are actually offered by the launcher.
> [`ffxiv-versions-sanity-check`](./sanity-check/) should ensure that the data is reasonably accurate.
